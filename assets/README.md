# Assets

Place final generated brand images here before release.

Expected files:

```txt
assets/branding/logo-mark.png
assets/branding/wordmark.png
assets/branding/app-icon.ico
assets/branding/installer-banner.png
assets/backgrounds/landing-background.png
```

Notes:

- `app-icon.ico` should be used for Windows installer/taskbar once available.
- `landing-background.png` is automatically referenced by the renderer stylesheet.
- If `landing-background.png` is absent, the app still uses a dark gradient fallback.
